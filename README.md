# Electronic-Engineering2

## Description:
This project controls 4 servo motors using Arduino UNO.  
The servos run using the Sweep example for 2 seconds,  
then all motors are held at 90 degrees.

##  Components:
- Arduino UNO
- 4x Micro Servo motors
- Jumper wires
- Power via USB
